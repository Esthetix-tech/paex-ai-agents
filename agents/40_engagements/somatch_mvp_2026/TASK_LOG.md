# SoMatch MVP 2026 Task Log

## 1. Purpose

This file tracks all tasks related to the `SoMatch MVP 2026` Engagement.

It is used to record:

- what needs to be done;
- which layer the task belongs to;
- who owns the task;
- current task status;
- evidence of completion;
- whether review is required;
- whether the task may produce reusable ESTRIX Core capability;
- what should happen next.

Core rule:

> A task is not complete until its output, evidence, layer classification and next action are clear.
